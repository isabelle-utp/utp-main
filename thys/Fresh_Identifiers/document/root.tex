\title{Fresh identifiers}
\author{Andrei Popescu \and Thomas Bauereiss}
\maketitle

\begin{abstract}
  This entry defines a type class with an operator returning a fresh
  identifier, given a set of already used identifiers and a preferred
  identifier.  The entry provides a default instantiation for any infinite
  type, as well as executable instantiations for natural numbers
  and strings.
\end{abstract}
