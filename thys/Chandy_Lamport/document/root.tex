\begin{abstract}
    We provide a suitable distributed system model and implementation the
    Chandy--Lamport distributed snapshot algorithm~\cite{chandy}. Our main
    result is a formal termination and correctness proof of the Chandy--Lamport
    algorithm and its use in stable property detection.
\end{abstract}
